# FastHire

FastHire — сервис поиска работы для IT-специалистов по механике взаимных матчей.

Кандидат:
- создает профиль
- получает вакансии
- ставит лайк

Работодатель:
- создает вакансию
- получает кандидатов
- ставит лайк

Если лайк взаимный:

MATCH

После матча открывается чат.

Основная цель:

Сократить путь от поиска до интервью до 15 минут.

# MVP Tasks

Версия: 1.0

Цель MVP:

Запустить первую рабочую версию FastHire с возможностью:

* регистрации пользователей
* создания профилей
* публикации вакансий
* свайпов
* матчей
* чатов
* базовой модерации

Без мобильного приложения.

Только Web.

---

# Этап 1. Инфраструктура

Статус: Критично

## Задачи

### Создать GitHub Repository

Сделать:

* README
* roadmap
* architecture
* database
* security

Результат:

Основа проекта готова.

---

### Создать Supabase проект

Настроить:

* PostgreSQL
* Auth
* Storage
* Realtime

Результат:

Готов Backend.

---

### Создать Next.js проект

Стек:

* Next.js
* TypeScript
* Tailwind
* Shadcn UI

Команда:

npx create-next-app@latest

Результат:

Готов Frontend.

---

### Настроить Vercel

Подключить GitHub.

Настроить автоматический деплой.

Результат:

Каждый push автоматически публикует сайт.
